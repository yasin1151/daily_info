
# GitHub Trending 今日 AI 项目

## 1. OpenInterpreter：面向低成本模型的本地 Coding Agent

项目简介：`openinterpreter/openinterpreter` 是一个用 Rust 构建的编码 Agent，定位是让 DeepSeek、Kimi、Qwen 等低成本模型也能承担代码执行和开发任务。项目 topics 包含 `coding-agent`、`acp`、`deepseek`、`kimi`、`qwen`。

为什么值得关注：它切中了当前 Agent 工具链的一个实际方向：不只绑定昂贵闭源模型，而是把 coding agent 能力下沉到更便宜、可替换的模型上。对于本地开发代理、低成本自动化编程、ACP 兼容生态都值得跟踪。

## 2. DeepTutor：长期个性化 AI 辅导 Agent

项目简介：`HKUDS/DeepTutor` 是一个 Python 项目，主打 lifelong personalized tutoring，结合 LLM、多 Agent、RAG、Deep Research 和交互式学习能力，面向个性化教育场景。

为什么值得关注：它不是单纯聊天式 tutor，而是把“长期学习画像 + 多 Agent + RAG”组合到教育工作流里。这个方向对 AI 教育产品、个人知识系统、长期记忆型 Agent 都有参考价值，尤其适合观察其个性化建模和学习路径生成方式。
