## 2. Code Review Graph：给 AI Coding 工具用的本地代码智能图谱

**项目简介**：`tirth8205/code-review-graph` 是一个 local-first 的代码智能图谱工具，面向 MCP 和 CLI。它会为代码库构建持久化结构图，让 AI 编程工具在代码审查和大仓库任务中只读取真正相关的上下文。
