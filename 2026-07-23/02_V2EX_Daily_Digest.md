3. 大家有用过 Power Automate 吗？

- 8 楼（点赞数：页面未公开）：用它自动处理公司 IT 密码流程，只需手机确认 2FA，随后自动获取密码并邮件发送，适合重复性内控流程。

4. 有没有什么好用的 SQL MCP 服务器

摘要：楼主想找 SQL MCP 服务器，让 AI 更方便访问数据库。评论核心观点是：数据库 MCP 价值有限，真正关键在权限隔离、危险操作限制和可审计执行，否则把数据库直接暴露给 Agent 风险很高。

高质量评论：
- 13 楼（8 赞）：质疑数据库 MCP 的必要性，认为很多 agent 已经会自己通过项目代码或连接方式执行 SQL。
- 12 楼（点赞数：页面未公开）：建议设置兜底策略，通过数据库用户权限控制危险操作，给 AI 使用的账号关闭 DELETE、DROP 等命令。
- 10 楼（点赞数：页面未公开）：认为数据库 MCP 多数时候是伪需求，因为代码中已有访问方式，模型通常会写脚本直接操作数据库。

原帖：https://www.v2ex.com/t/1228897

5. [慢迅] Surge 6 支持 tailscale 了

摘要：Surge 6 支持 Tailscale 引发网络工具链讨论。价值点在于移动办公和家庭内网访问的工程方案：Tailscale、WireGuard、Surge 白名单、TCP/UDP QoS 都会影响稳定性和可维护性。

高质量评论：
- 3 楼（8 赞）：WireGuard 走 UDP，很多运营商会对 UDP 做较高级别 QoS；Tailscale 走 TCP 时可能更稳定。
- 1 楼（点赞数：页面未公开）：长期使用 Surge + Tailscale 常驻，遇到 Tailscale 网段进 Surge 报错页时，可通过 Surge 白名单解决。
- 2 楼（点赞数：页面未公开）：也可以在家里部署 easytier 并启用 WireGuard，Surge 配 WireGuard，实现自动连回内网，VPS 也能串联。

原帖：https://www.v2ex.com/t/1229110
