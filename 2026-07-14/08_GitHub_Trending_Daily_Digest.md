**1. Airi：自托管 AI Companion / Agent 容器**  
项目简介：`moeru-ai/airi` 是一个自托管、用户自有的 AI companion 项目，支持实时语音聊天，并可扩展到 Minecraft、Factorio 等交互场景，支持 Web、macOS、Windows。  
为什么值得关注：它不是单纯聊天 UI，而是把语音交互、角色化 Agent、桌面端和游戏环境连接在一起，适合观察“个人 AI Agent / Companion”产品形态。  
Stars：41,847，今日新增 57  
链接：https://github.com/moeru-ai/airi

**2. Graphify：把代码库转成可查询知识图谱的 AI 编码助手技能**  
项目简介：`Graphify-Labs/graphify` 面向 Claude Code、Codex、OpenCode、Cursor、Gemini CLI 等 AI 编码工具，可把代码、SQL schema、脚本、文档、论文、图片或视频转成可查询知识图谱。  
为什么值得关注：代码知识图谱正在成为 AI 编程工具的重要上下文层。这个项目覆盖代码、数据库 schema 和基础设施，方向上贴近大型代码库理解、Agentic coding 和企业级代码检索。  
Stars：84,641，今日新增 1,028  
链接：https://github.com/Graphify-Labs/graphify

**3. Spec Kit：GitHub 推出的规格驱动开发工具包**  
项目简介：`github/spec-kit` 是 GitHub 发布的 Spec-Driven Development 工具包，用于帮助项目从规格说明开始组织开发流程。  
为什么值得关注：规格驱动开发正在和 AI 编程工作流结合得越来越紧密，尤其适合减少“直接让 Agent 写代码”带来的需求漂移。GitHub 官方项目本身也值得跟踪其后续生态影响。  
Stars：120,561，今日新增 508  
链接：https://github.com/github/spec-kit
